# CAD-Training

A curated portfolio of 3D models, components, and assemblies developed as part of my self-directed engineering design practice. This repository showcases a mix of personal projects, practical problem-solving tools, and exploratory drafting concepts.


### Technology Stack
* **Software:** Autodesk Inventor
* **File Formats Included:** Native CAD Parts (`.ipt`) and standard 3D printing/preview meshes (`.stl`)


### Scope of Work
This collection represents the progression of my modeling capabilities, ranging from foundational early exercises to functional assemblies designed for real-world utility. 

While these models focus primarily on functional utility and geometric experimentation rather than strictly managed commercial standards, they demonstrate a solid grasp of:
* Parametric part design and sketching hierarchy
* Assembly constraints and spatial awareness
* Design iteration and rapid prototyping workflows


### Downloading
As for downloading, you are free to download any available models within the repository. Simply navigate to the `Models/` directory and select any model that you might be interested in. Please note that GitHub does not natively display previews for these file types, but the repository is structured so that the file names themselves are highly descriptive and self-explanatory. 

In the future, I plan to bundle these assets into dedicated folders containing the model files, preview images, and assembly files for projects that require assembly. Additionally, I will include a simple descriptive text document within these model folders to provide more context and explain the background of each model. While these models represent a continuous learning process rather than high-end production assets, I am committed to making them as clean and professional as possible as I continue to learn and familiarize myself with the software.
